Publishing With GitHub Pages

This site is plain static HTML and CSS. No build step is required.

Repository Settings

  1. Push the repository to GitHub.
  2. Open the repository Settings.
  3. Go to Pages.
  4. Set the source to “Deploy from a branch.”
  5. Select the branch you want to publish from.
  6. Set the folder to /docs.
  7. Save.

GitHub will publish docs/index.html as the site home page.
